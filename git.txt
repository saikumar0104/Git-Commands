--------------------------------------------------------------------------------
git init	                                        initializes  a new Git repository
git config --global user.name "Your Name"	        Set your username (used in commits).
git config --global user.email "you@example.com"	Set your email address.
git config --list                                	Show all Git configuration settings.
git help <command>                              	Show help for a specific Git command.

git init	                                        Initialize a new local Git repository.
git clone <url>                                  	Clone a remote repository to your local machine.

git status	                                        Show the current status of your working directory.
git add <file>	                                    Stage a specific file for commit.
git add .	                                        Stage all changed files.
git commit -m "commit message"	                    Commit staged changes with a message.
git commit -am "commit message"                    	Add and commit all tracked files in one step.
git diff	                                        Show differences between working directory and staging area.
git diff --staged	                                Show differences between staging area and last commit.

git branch	                                        List all branches.
git branch <name>	                                Create a new branch.
git checkout <branch>	                                Switch to another branch.
git checkout -b <branch>                        	Create and switch to a new branch.
git merge <branch>                              	Merge a branch into the current branch.
git branch -d <branch>	                                Delete a branch.

Note: one extra commit is added in merge as compared to rebase

git remote -v	                                        Show remote repositories.
git remote add origin <url>                       	Add a remote repository.
git push -u origin <branch>	                        Push a branch to a remote repository (set upstream).
git push	                                        Push committed changes to a remote repository.
git pull	                                        Fetch and merge changes from a remote repository.
git fetch                                       	Download changes from a remote repo but don’t merge.

git log                                          	Show commit history.
git log --oneline	                                Show compact commit history.
git show <commit>	                                Show details of a specific commit.
git blame <file>	                                Show who changed each line in a file.

git restore <file>                              	Restore a file from the last commit.
git reset <file>                                  	Unstage a staged file.
git reset --hard	                                Reset to last commit, discarding all changes.
git revert <commit>                             	Create a new commit that undoes a previous commit.

git rebase main	                                        Rebase current branch on top of main.
git rebase -i HEAD~3                            	Interactive rebase for last 3 commits (edit, squash, reword, etc.).
git rebase --continue                           	Continue rebase after resolving conflicts.
git rebase --abort                               	Cancel a rebase in progress.

git diff main origin/main                        Shows diff b/w main and branch
git diff master branch
